# AWS SOC Monitoring Lab

## Overview

AWS SOC Monitoring Lab is a cloud security monitoring project using AWS IAM, CloudTrail, S3, and MFA to monitor account activities, analyze authentication events, and investigate security logs within an AWS environment.

## Services Used

* AWS IAM
* AWS CloudTrail
* Amazon S3
* MFA Authentication

## Features

* IAM user configuration
* MFA setup
* CloudTrail monitoring
* Failed login investigation
* Event log analysis

## Project Workflow

IAM User → CloudTrail → S3 → Security Investigation
